Connect to a USB printer:
1
Connect the USB printer to the USB port of the router.
2
Download and Install the driver software of the USB printer on each computer where you want to use the printer.
If you have no driver software of the USB printer, please contact the manufacturer of the printer to provide the latest driver software.
3
Download the USB Controller utility program from Tenda website (www.tendacn.com) and install it on each computer where you want to use the printer. Use the USB Controller utility program to establish a connection between the computer and the USB printer according to the instruction of the utility program.
