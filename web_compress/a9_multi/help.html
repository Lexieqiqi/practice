Connection Wizard
Back to Contents

The first time you setup the Essentials Wi-Fi Extender the connection wizard will help guide you through connecting the Wi-Fi Extender with your broadband router/hub.

Connection Wizard: Step 1 (Select Wi-Fi Name)
Back to Contents

This screen lets you select the Wi-Fi Name of the broadband hub/router you wish to connect to. After selecting the Wi-Fi Name click  Next  to continue.
If your hub/router does not appear in the list, click on the  Refresh  icon to update the network list.

Connection Wizard: Step 2 (Enter Password)
Back to Contents

To connect the Essentials Wi-Fi Extender to your selected broadband hub/router, enter the Wi-Fi Password for the broadband router, then click  Finish.

When you click Finish, you’ll lose connection to your browser; it may display a network unavailable error which you can ignore. When the Wi-Fi Extender connects to your hub/router it will switch off its default extender Wi-Fi name (BTWifiExtndr). So if your device was connected to this Wi-Fi Name for setup purpose, the connection will end.

Your devices can now use the Wi-Fi extension using the new network name and key as your main broadband hub/router.

Connected Devices
Back to Contents

This section shows all the devices that are connected to your Essentials Wi-Fi Extender.

When you first open this page, the Wi-Fi Extender will scan your local network to identify all devices connected to it. Each device is listed with its MAC Address to uniquely identify them; the MAC address is normally found on labels on products. The Device Name is a name built into the device and can't be changed on the extender; in some cases it may even be blank if the manufacturer hasn't set one up.

Setup

This shows the wireless details of your connection back to your broadband hub/router.

The Connection Status and Connection Time are displayed here.

You can also change the broadband hub/router that you are connected to. Select the Wi-Fi Name from the list of Wi-Fi networks.

If your hub/router does not appear in the list, click on the  Refresh  icon to update the network list.

To complete the connection of the Essentials Wi-Fi Extender to your selected broadband hub/router, you will be asked to enter the Wi-Fi Password for the broadband router. Click  OK  to finish.

WPS (Wi-Fi Protected Setup) Settings
Back to Contents

Your Essentials Wi-Fi Extender has a Wireless WPS (Wi-Fi Protected Setup) button which supports Push Button to Connect.

The WPS button on the front of the device can only be used to connect the Essentials Wi-Fi Extender to your hub/router.

To find out more about WPS, go to  www.wi-fi.org/discover-wi-fi/wifi-protected-setup.

Extended Network
Back to Contents

This section allows you to change the Wi-Fi Name (SSID) of your Essentials Wi-Fi Extender.

In most cases, you shouldn't need to change these settings. By default your Wi-Fi Extender will relay the wireless settings from your main broadband hub/router with prefix 'EXT-' added to the router’s Wi-Fi Name so your devices can benefit from the extension.

You may choose to change these settings if you want the extended network to use a different network name; however under normal use you shouldn't need to do this.

If you make changes to the Wi-Fi Name, ensure you write them down as you will need to enter them on any devices you wish to connect.

You can restore the default settings by using the "Reset" pinhole on the bottom of the Wi-Fi Extender. (Use a paper clip or other thin object and press the button for 1 second)

Extender Wi-Fi Name

This is the name that is broadcast to identify your Essentials Wi-Fi Extender for connecting new devices.

The default Wi-Fi Name/SSID is unique and set to BTWifiExtndr-XXX (where X are random alphanumeric characters).

You can change the Wi-Fi Name using up to 32 alphanumeric characters (case-sensitive).

Advice for choosing a Wi-Fi Name/SSID

  • Characters ? " $ [ \ + aren't allowed;
  • Name can't start with ! # ;
  • No <space> at start or end;
  • Name can't start with the text 'DIRECT' (This name may conflict with future services);
  • Name can’t contain the text "BTWifi", "BTWi-fi", "Auto-BT", "BTOpenzone" or "BTFON".

Extender Wi-Fi Password

This is the password that is used to connect new devices to the Essentials Wi-Fi Extender.

Advice for a secure Wi-Fi Password

To help make your Wi-Fi Password as secure as possible, please make sure that it:

  • is at least 10 characters long, but no more than 63
  • is a combination of numbers, upper-case letters (or 'CAPITALS'), lower-case (or 'small') letters
  • does contain punctuation, but no spaces
  • is not a name or a dictionary word
  • does not repeat the same character one after another (e.g. aabbccdd00)
  • is not a commonly used key like 'abcd1234', 'welcome1', etc
  • is not the same as information given elsewhere (e.g. your Admin password, serial number, MAC address or wireless network/SSID)

Advanced
Change Admin Password
Back to Contents

The default Administration (Admin) Password of your Essentials Wi-Fi Extender has been generated randomly and is unique to your device.

If you wish to change the password:

  • Enter the  Old Password
  • Enter the new password twice, once in  New Password  and again in the  Confirm New Password  fields
  • Click  Save

Please note the following points about your new Admin password:

  • The new password can be up to 20 characters long, but for increased security, it should be as long as possible (at least 8 characters long), contain a mix of letters, numbers and capitalisation, and should also not include dictionary words or common names.
  • Your password is case sensitive
  • If you change your password, keep a note of it somewhere safe

If you forget your password, you can restore the default value by using the reset button on the bottom of your Wi-Fi Extender. Please note, pressing the reset button will restore all settings (e.g. Wi-Fi Name/SSID) to their default values, and you'll need to reconnect to your broadband hub/router.

You will have 5 attempts to enter the password correctly. If you do enter it incorrectly 5 times, you will be locked out of Extender for 1 minute. If that happens, close your browser window and try again 1 minute later. Should the password still be incorrect on a subsequent attempt, the amount of time you'll be unable to access Extender will increase until it reaches 15 minutes. At that point you won't be able to enter a password for 15 minutes until it is entered correctly. Turning your Wi-Fi Extender off and on will reset this counter and allow you to have 5 consecutive attempts again.

Once you click 'Save' to change your password

The new setting takes effect immediately. When you continue to access other pages, you may be prompted to log in again with your new password.

Reboot Extender
Back to Contents

You can reboot the Essentials Wi-Fi Extender from within the management interface, simply click the Reboot button. You will be ask for confirmation before the reboot is carried out.

All devices currently connected to the Wi-Fi Extender will lose their connection until the device has restarted.

Reset to Factory Defaults
Back to Contents

If you wish to restore your Essentials Wi-Fi Extender to the original settings, you can do this by using the reset button on the bottom of your Wi-Fi Extender. Please note, pressing the reset button will restore all settings (e.g. Wi-Fi Name Wireless Network/SSID) to their default values, and you'll need to reconnect to your broadband hub/ router.

Firmware Upgrade
Back to Contents

Firmware updates may be released periodically to improve the functionality of your Essentials Wi-Fi Extender. Your current firmware version is shown, if you experience problems with your device please check if there is an update available on the BT support website  bt.com/essentials300.

To update the firmware in your Essentials Wi-Fi Extender:

  • First, download the new firmware to your computer (the file will have an .img extension)
  • If you wish to retain any settings you have changed, please ensure the Clear Config option is unticked
  • Click the browse button and select the new firmware
  • The upgrade will now complete.

The update takes approximately 2 minutes after which your Wi-Fi Extender will reboot. Please do not turn off your device until the update is complete.
